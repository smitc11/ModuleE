# Electricity Price Forecasting & Market Trends (Colab Notebook)

This repository contains a **Google Colab–based Jupyter Notebook** for analyzing **electricity price forecasting and market trends**, using a synthetic SCADA-style dataset.

The project is designed to be:
- Easy to run in **Google Colab**
- Cleanly structured for **GitHub review**
- Suitable for **data science / ML portfolio use**
